# Cluric

### A python tool made for check Username from over 100 Popular websites.


## Usage
`apt-get update` <br>
`apt-get upgrade` <br>
`pkg install git` <br>
`pkg install python3` <br>
`git clone https://github.com/thenurhabib/cluric` <br>
`pip install requests` <br>
`cd cluric` <br>
`.python3 cluric.py`
