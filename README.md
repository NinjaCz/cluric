# Cluric

### A python tool made for check Username from over 100 Popular websites.

 ```yaml

apt-get upgrade
pkg install git
pkg install python3
git clone https://github.com/thenurhabib/cluric
pip install requests
cd cluric
python3 cluric.py
```
